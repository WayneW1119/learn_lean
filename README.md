# Lean Kernel Model

Versioned documents modeling the Lean kernel and elaboration layer, from v0 to v10.

Each version builds on the previous one, progressively adding features:
- **v0**: Core terms and basic kernel
- **v1-v8**: Incremental refinements
- **v9**: `StructureCmd` — all structures defined via command
- **v10**: `ProjectionInfo`, `SProj`, `SDot`, projection elaboration

Each document is self-contained but highlights only differences from the previous version.
